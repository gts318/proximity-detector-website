Proximity Detector
  • Function
  • Components
  • Schematic
  • Code

The purpose of this device is to detect close objects and alert the user using lights and sound. An ultrasonic sensor is used to detect objects in front. When an object comes within one meter of the sensor, the LEDs will light and the buzzer will sound. As the object approaches, more LEDs will light and the pitch of the sound of the buzzer with become higher. Each unlit LED represents 12.5 centimeters of space between the sensor and the approaching object.
